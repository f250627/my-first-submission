# Signature Cuisine Website

This is my web development project for ICT.  
It includes a homepage, menu page, contact page, and other basic site features.

## Project Overview
The website is designed for a fictional restaurant called **Signature Cuisine By ADH**.  
It contains:
- A hero section with welcome text
- A menu button
- Navigation bar with links
- Multiple sections with content and images
- A responsive layout

## Files Included
- `index.html` or `home.html` – Homepage of the website
- `menu.html` – Menu page
- `about.html` – About Us page (if included)
- `contact.html` – Contact page
- `style.css` – Main CSS file
- `script.js` – JavaScript file
- `/images/` – Folder containing images used in the site

## Features
- Responsive navigation bar
- Embedded YouTube video
- Background images
- Buttons and hover effects

## Technologies Used
- **HTML**
- **CSS**
- **JavaScript**

## How to Run the Project
1. Download or clone all project files.
2. Keep all files in the same folder structure.
3. Open `home.html` (or `index.html`) in any browser.
